# simulazione-moto_parabolico
Capolavoro 2025/2026

Il programma consiste in un software di simulazione fisica sviluppato per visualizzare il moto di un proiettile. L'utente può inserire parametri come la posizione iniziale, l’angolo di lancio, la velocità e l’accelerazione di gravità. Il programma calcola e visualizza tramite un grafico la traiettoria, il tempo di volo, la gittata massima e l'altezza massima, permettendo di analizzare come le variabili fisiche influenzano la traiettoria del proiettile.
